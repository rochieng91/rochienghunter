# Deriv Digit Match Prediction Application

## Overview
The Deriv Digit Match Prediction Application is designed to predict the outcomes of digit match games on the Deriv platform. This application leverages machine learning models and historical data to provide insights on potential match results.

## Requirements
- Python 3.x
- pip (Python package manager)
- Required Python packages listed in `requirements.txt`

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/rochieng91/rochienghunter.git
   cd rochienghunter
   ```
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Configuration
1. Create a `.env` file in the root directory of the application.
2. Add the following configuration variables:
   ```env
   DERIV_API_KEY=your_api_key
   LOG_LEVEL=info
   ```
   Replace `your_api_key` with your actual API key from Deriv.

## Usage
To run the application, execute the following command in the terminal:
```bash
python main.py
```
